# settle

`settle` is a CLI tool that helps you manage your digital Zettelkasten.

## Getting started

### Requirements

* cargo/rust toolchain (for building)
* SQLite (for running)

### Installation

There's a Rust crate available, so you can simply run:

```
cargo install settle
```
